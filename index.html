Project Background

Modern vehicles now incorporate advanced sensors, cameras, and by-wire technology to enable new autonomous driving functions. Parking remains a common challenge for many drivers, often causing stress and increasing the chance of mistakes. By leveraging existing vehicle technologies, an automated parking system can enhance safety, convenience, and overall operational efficiency. The Active Park Assist system provides vehicles with self-parking capability, requiring only minimal driver input to park in either parallel or perpendicular spaces.

Project Description

The Active Park Assist system enables vehicles to perform automated parking maneuvers within designated parking areas. Through its Human-Machine Interface (HMI), drivers can select the type of parking and confirm available spaces. Once engaged, the system assumes control of vehicle acceleration, braking, steering, and transmission gear shifts to execute safe and precise parking operations. To prevent collisions, the system continuously monitors the vehicle’s surroundings using cameras, radar, and ultrasonic sensors. Drivers can regulate speed using the brake pedal or access vehicle controls remotely through a mobile application. The Active Park Assist system operates through several integrated subsystems—including Park Control, Powertrain Management, Brake Control, Steering Control, and Vehicle Positioning—which work together to ensure reliable and accurate parking performance, even in the event of sensor failures or security threats.
